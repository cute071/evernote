# 自煮-炒烏龍麵


1. 肉肉先用酒醬油醃一下（我也忘了哈哈）
2. 香菇泡水剪成條狀
3. 爆香蒜頭洋蔥、加入高麗菜炒一炒，撥到旁邊再加肉下去炒混合後加醬油胡椒香菇水
4. 再加烏龍麵！上面寫不用退冰我就直接放了！
5. 其實水可以再加！然後沙茶放入
6. 醬油可以再添加多一點 因為我以為醃肉就會鹹了結果根本不夠～多加了好幾次～再加點白胡椒 好香
7. 完成✅呈盤後覺得少一味 就加點烏醋 好吃耶！別家太多 會有提味感 甜甜的。再加典好吃的泡菜！要放在麵裡悶熱一下比較好吃不會太辣！
