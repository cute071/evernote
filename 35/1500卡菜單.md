# 1500卡菜單


女生版 (健康減肥的話，每日飲食熱量最低1500卡)           用餐時間
第一餐：早餐 500卡  全麥饅頭、豆漿、花生醬                  7-8點
第二餐：點心 80卡    蘋果                                                    10-11點
第三餐：午餐 500卡  各種蔬菜、蛋、魚肉、飯                  12-1點
第四餐：點心 80卡    奇異果                                                 3-4點
第五餐：晚餐 300卡  冬粉、海帶芽、蕃茄、海鮮               6-7點
第六餐：點心 80卡    熱牛奶                                                  9-10點

女生版 (遇到停滯期時，每日飲食熱量最低1200卡)          用餐時間
第一餐：早餐 500卡  全麥饅頭、豆漿、花生醬                  7-8點
第二餐：點心 80卡    蘋果                                                    10-11點
第三餐：午餐 400卡  各種蔬菜、蛋、魚肉、半碗飯          12-1點
第四餐：點心 80卡    奇異果                                                 3-4點
第五餐：晚餐 200卡  冬粉、海帶芽、蕃茄                           6-7點
