# 桂圓紅豆紫米粥


桂圓紅豆紫米粥的食材和调料：

　　1. 紅豆 半碗

　　2. 紫米 半碗

　　3. 水十碗

　　4. 冰糖 適量

　　5. 乾桂圓 半碗

　　6. 小湯圓或是小芋頭丸子 一碗

　　教您桂圓紅豆紫米粥的家常做法，如何做桂圓紅豆紫米粥才好吃

　　1. 紫米加紅豆加水用壓力鍋煮至軟，差不多20 分鐘左右。

　　2. 待壓力減去，開鍋，加入冰糖和乾桂圓，用一般煮法（無壓力）小火慢煮40分鈡。如果太濃稠，就加奌水。

　　3. 加入小湯圓或芋頭丸子，待湯圓煮熟浮起，好吃的紫米粥就完成了。
