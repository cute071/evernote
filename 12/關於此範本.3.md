關於此範本
壓力雖然，但壓力過大、時間過長就可能導致嚴重倦怠、焦慮等各種健康問題。妥善處理壓力的第一步就是能自覺。一週兩三次，特別是在感覺到壓力很大時，可以使用下面的評估表作評估。1 分是幾乎無感，10 分是非常有感，看看每項得到幾分。有了這樣的分析記錄，您就可以更有效的掌握自己的壓力程度。

| YYYY.MM.DD |     |     |     |
| 感覺壓力 |     |     |     |     |
