也來分享在留學生界很紅的快煮辦法， 需要一個10份大同電鍋， 以及6個筒仔米糕的鐵筒。 事前準備： 1、一次煮好4-5餐的白飯，分裝小包。 2、薑絲、蒜片、蔥花，備好放保鮮盒。 3、魚、蝦肉片等，分裝冷凍。 4、枸杞紅棗蔘片，隨意。 （以上，可在週末備好） 煮食： 1、一筒內裝白飯。 2、雞蛋+雞精毛豆仁等，一筒蒸蛋。 3、一種青菜，二種也可以。 4、一片魚+薑絲、豆豉，一筒蒸魚。 5、肉片+金針菇+豆腐亂加，一道雜炊。 6、薑絲蛤蠣、香菇雞、枸杞紅棗魚，一道湯。 電鍋按下去，20分鐘後四菜一湯，開飯！ ※ 引述《thiopento (清者自清濁者自濁)》之銘言： : 我之前是一打二喔! : 大的是1y10m發展遲緩無口語，現一個人二隻手一大一小帶到小的1y8m重回職場 : 一開始坐月子是自己煮，媽媽幫我顧一下孩子 : 買網路上的中藥包30包，每天就是拿一包藥+一片梅花肉排燉一小鍋湯 : 炒一盤青菜和一鍋飯就解決了 : 後來媽媽在我坐完月子就回家了 : 我就請婆婆每天煮飯時多煮一份，冷凍起來 : 每隔1~2週幫我寄冷凍宅配 : 所以每天晚餐就是開個1-2包回鍋加熱 : 主要是大寶吃，我吃他吃剩的 : 有多的就給老公帶便當隔天中午繼續吃 : 後來我發現大寶不喜歡婆婆煮的飯 : 我只好用背巾後背著二寶下廚，因為他一定要粘在我身上不願意躺安撫椅 : 煮飯的話就是盡量簡單、營養(蛋白質、青菜、澱粉一定要同時有) : 不過我大部分還是為了大寶煮的，中午忙碌到自己根本無法進食 : 所以產後快速消瘦 : 不過我後來訓練到快手煮飯，從二寶開始副食品就是跟著大人上桌吃了 : 所以我晚餐就是大寶在客廳看卡通，帶著二寶進廚房開伙 : 一家四口的晚餐加爸爸隔天中午的便當大約一個小時搞定 : 推薦你幾個簡單的方法： : 1.電器配備要齊全 : 我下廚的時候一定是開四種電器；電子鍋煮飯、電鍋燉湯、烤箱烤魚、瓦斯爐炒菜 : 這樣才能幫你有效的節省時間 : 2.常備好湯 : 通常是買好市多的排骨，夏天燉筍子排骨湯、冬天燉蘿蔔排骨湯 : 電鍋丟著慢慢燉不用顧火，安全又方便 : 一鍋湯大概喝二天，超過就倒掉 : 這樣你的餐桌上就有基本菜色：蔬菜(蘿蔔或筍子)、蛋白質(排骨)、高湯 : 隔天中午可以利用這鍋湯底煮個麵、冬粉、麵線 : 3.善用電鍋 : 電鍋的功能實在太強大，所以我白飯一律用電子鍋煮，我買日本進口的虎牌電子鍋 : 和日本進口米，所以白飯是基本很好吃的 : 電鍋的話就拿來燉湯，可以再加架一個蒸盤蒸蛋、蒸花椰菜高麗菜、蒸回鍋菜 : 4.食材品質 : 因為自己開伙加上媽媽我本身很辛苦，我在選擇食材上會非常講究 : 由於好的食材並不需要過度烹調，簡單水煮、清蒸都很好吃 : 5.善用冷凍食材、方便食材 : 我會用韓式燒烤醬+洋蔥+牛肉炒一盤牛肉，大人小孩都愛那口味，非常好吃 : 然後傳統超市可以買到打拋醬，一包白色的泰國進口，用牛豬絞肉可以快炒一盤 : 我會去買有機店的冷凍花椰菜，只要稍微加熱就可以吃了 : 買有機店的鮪魚肉醬，有時候沒時間煮，水滾個麵加個青菜肉醬就是一餐 : 可以買真空包裝的無刺虱目魚肚常備冷凍庫，水煮、不沾鍋乾煎、烤箱烤都好吃 : 還有推薦義美的蔥油餅，有時沒胃口煎個蛋灑上黑白芝麻也很好吃 : 另外我的冷凍庫一定會有吻仔魚，煮粥、煎蛋都很適合 : 還有還有，我冰箱也都會有冷凍的玉米粒，通常是綠x人，開封之後冷凍起來 : 有時灑在湯裡可以幫助降溫，小朋友很喜歡 : 另外也有買韓國海苔，沒胃口的時候就拌白飯吃或做成飯團 : 6.多強化一些工具 : 像是十字花科的蔬菜比較會有洗不乾淨的疑慮 : 所以我上網去買一台攜帶式的超音波清洗器，把菜先用超音波洗個5-10分鐘左右 : 那時候我可以同時做別的事 : 吃完的碗盤也可以用他幫忙先清洗一下，自己煮通常不會太油膩 : 震個幾下有時就乾淨了 : 另外是家裡若有雙口瓦斯爐其中一口可以用鑄鐵鍋燉肉 : 然後烤箱是一個很棒的工具! : 盡量買小一點，大約6-9L，加熱速度快，烤魚只要抹完鹽巴不用預熱烤15-20分鐘 : 氣炸鍋也是個不錯的工具，拿來炸四季豆、香腸都很棒喔 : 再來是我有一台食物處理器 : 我會利用週末有隊友在時，把紅蘿蔔、南瓜一次用機器刨成絲，分成小包冰在冷凍 : 需要的時候折一小瓣，煮麵、快炒都可以 : 這些是我一打二的小撇步 : 提供給你參考，希望對你有幫助 : ※ 引述《agnel (不是天使?
